# Skills

My personal collection of reusable agent skills.

## Available Skills

### agents-md-curator

`agents-md-curator` helps coding agents generate, audit, and refine `AGENTS.md` files for code repositories. It focuses on high-signal repository guidance: exact build/test commands, non-obvious conventions, validation expectations, and safe boundaries for agentic code changes.

Install this skill with:

```bash
npx skills install lifanh/skills/agents-md-curator
```

Skill path:

```text
agents-md-curator/SKILL.md
```
